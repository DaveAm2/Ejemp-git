Indicaciones:
1. El proyecto será desarrollado con Python.
2. Se deberá crear un entorno virtual para instalar dependencias
        python -m venv venv
3. Ejecutar el comando para instalar dependencias descritas en el 
   archivo requirements.txt
        pip install -r requirements.txt
4. El comando para verificar la dependencias instaladas es:
        pip freeze
5. El comando para guardar en un archivo de texto las dependencias
   necesarias para el proyecto es:
        pip freeze>requirements.txt
